# PriceScout Project 🚀

Асинхронный конвейер для парсинга, очистки и сохранения данных о товарах. Проект построен на микромодульной архитектуре с соблюдением принципа разделения обязанностей

## 🛠️ Технологический стек
* **Язык:** Python 3.10+
* **Асинхронность:** `asyncio`, `aiohttp`
* **База данных:** SQLite (асинхронный драйвер/модели)
* **Инструменты:** `fake-useragent`

## 🏗️ Архитектура проекта
Проект разделен на изолированные слои, что позволяет легко менять источник данных (парсер) или хранилище (БД) без изменения бизнес-логики:

* `core/base_parser.py` — Базовый абстрактный класс для всех будущих парсеров.
* `core/uzum_parser.py` — Модуль сетевого парсинга (настроен на работу с веб-ресурсами / Fake Store API для безопасного тестирования).
* `modules/processing.py` — Бизнес-логика: фильтрация «грязных» данных, проверка цен и валидация через генераторы (`yield`).
* `database/models.py` — Слой хранения: инициализация таблиц и сохранение объектов товаров.
* `main.py` — Главный оркестратор (дирижёр), связывающий все компоненты в единый асинхронный конвейер.
